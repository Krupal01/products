# 📞 Auto Redial – Smart Automatic Calling Assistant  

Auto Redial is a powerful and user-friendly calling automation app designed to help you reconnect effortlessly. Whether you're trying to reach a busy customer support line, participate in contests, or repeatedly dial an important number, Auto Redial handles the process for you — efficiently and reliably.

Built with simplicity and performance in mind, Auto Redial gives you full control over call frequency, intervals, SIM selection, and scheduling — all through a clean and intuitive interface.

## 🚀 Features

### 🔁 Multiple Call Attempts
Set the exact number of times you want to redial. No more manual retries — let the app handle it automatically.

### ⏱ Custom Call Interval
Choose the delay (in seconds) between each call attempt. Control the pace based on your needs.

### 📅 Schedule Calls for Later
Plan your calls in advance by selecting a specific date and time. Perfect for reminders or timed dialing.

### 📲 SIM Selection
Select your preferred SIM card for outgoing calls (useful for dual-SIM devices).

### 🧩 Compact Overlay Mode
Enable compact overlay to monitor and control call attempts while using other apps.

### 📊 Live Call Status
Track remaining calls, interval timing, and real-time countdown with a clean visual timer interface.

### ⚡ One-Tap Start
Simply enter a phone number, adjust settings, and tap **Start Dialing** — it's that easy.
